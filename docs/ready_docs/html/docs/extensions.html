Extensions¶

File containing all supported extensions, assets names and other static informations.

Sublime Text 3 informations¶

Default text to display and icon file name.

sublime_text = {
        "logo": "st3_logo",
        "header": "Sublime Text 3"
}

Unknown file informations¶

Text and icon file name designed when unknow file occured.

unknown = {
        "icon": "unknown_file",
        "header": "Unknown file"
}
